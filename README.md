# PI3-SportData-Insight
**Projeto Integrador III | Entrega 2**

## 👤 Identificação
*   **Nome:** Amon Brandão Lares
*   **Turma:** 5SC1
*   **Tema:** Análise da discrepância entre estilo de vida ativo e baixa participação em eventos esportivos universitários.

## 📝 Descrição do Projeto
Este projeto utiliza Ciência de Dados para investigar por que alunos que possuem hábitos saudáveis (como musculação e corrida) não estão engajados nos eventos esportivos da universidade. O objetivo é fornecer insights para atléticas e instituições aumentarem a integração social.

## 🛠️ O que foi entregue (Entrega 2)
1. **Análise de Dados Preliminar:** Scripts em Python realizando a limpeza e exploração inicial dos perfis estudantis.
2. **Modelo de Machine Learning:** Implementação do algoritmo de **Clusterização K-Means** para segmentar os alunos em 3 perfis (Engajados, Atletas de Academia e Desconectados).
3. **Protótipo de Visualização:** Estrutura de Dashboard para tomada de decisão baseada em dados.
4. **Vídeo de Demonstração:** Apresentação do protótipo e explicação dos insights gerados.

## 📊 Tecnologias Utilizadas
*   Python 3.11
*   Pandas & NumPy (Manipulação de dados)
*   Seaborn & Matplotlib (Visualização)
*   Scikit-Learn (Clusterização)
